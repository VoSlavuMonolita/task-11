# task-11
предметный указатель. Каждый объект содержит в себе слово и номера страниц на которых оно встречается. Реализованы возможность создавать указатель из файла,  сохранять его в XML-файл, добавлять, удалять, обновлять и искать информацию об объектах указателя, возможность печати указателя и вывода номеров страницы для заданного слова.
